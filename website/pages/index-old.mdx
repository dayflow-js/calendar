## Key Features

### Plugin Architecture

Extend functionality with a powerful plugin system. Add drag-and-drop, custom event handlers, or integrate with your existing tools.

```tsx
import { useCalendarApp, createDragPlugin } from 'react-calendar-package';

const dragPlugin = createDragPlugin({
  enableDrag: true,
  enableResize: true,
  enableCreate: true,
});

const calendar = useCalendarApp({
  plugins: [dragPlugin],
  // ... other config
});
```

### Flexible Event Handling

Full control over how events are displayed and interacted with. Customize event appearance, detail panels, and user interactions.

```tsx
import { DayFlowCalendar, useCalendarApp } from 'react-calendar-package';

function MyCalendar() {
  const calendar = useCalendarApp({
    events: myEvents,
    onEventClick: event => console.log('Clicked:', event),
    onEventUpdate: event => console.log('Updated:', event),
  });

  return <DayFlowCalendar calendar={calendar} />;
}
```

### Modern Temporal API

Built on the modern Temporal API for robust date/time handling, ensuring accurate timezone support and internationalization.
