# Features

Welcome to the DayFlow features showcase. Each page demonstrates a key capability with live, interactive examples:

- **View Switcher Modes** – compare button and select based switchers side by side.
- **Custom Event Detail Content** – render business data and actions inside the default panel.
- **Custom Event Detail Dialog** – launch a fully bespoke modal for event details.
- **Event Dialog** – built-in event dialog with create, edit, and delete functionality.
- **Sidebar** – explore the sidebar component for additional event management.

Pick a topic from the sidebar to start exploring.
