# モバイル対応

DayFlow は、モバイルデバイス向けに組み込みのサポートを提供します。カレンダーは、小さな画面でスムーズな体験を提供するために、レイアウトとインタラクションを自動的に調整します。

## レスポンシブレイアウト

すべてのビュー（日、週、月）はレスポンシブに設計されています：

- **月ビュー**: イベント時間を非表示にし、コンパクトな「他のイベント」インジケーターを使用します。また、読みやすさを最大化するために、イベントタイトルのグラデーションフェードによる切り詰めを採用しています。
- **週・日ビュー**: 狭い幅に収まるように、時間軸の列を自動的に縮小し、日付ラベルを簡略化します。
- **サイドバー**: サイドバーはモバイルではスペースを節約するために自動的に折りたたまれますが、トグルボタンからアクセス可能です。
- **ヘッダー**: 表示切り替え（View Switcher）は、混雑を避けるためにモバイルでは自動的に `select` モードに切り替わります。

## モバイルでの操作

DayFlow は、イベントの管理を容易にするためにモバイル特有の操作を導入しています：

- **長押しで移動**: モバイルでは、イベントを 500ms 長押しするとドラッグ＆ドロップによる移動が開始されます。
- **リサイズインジケーター**: イベントが選択されると（長押しまたはタップ）、角に大きなタッチフレンドリーなハンドルが表示され、サイズ変更が容易になります。
- **イベントドロワー**: イベントをタップすると、ボトムシート形式のドロワーが開き、詳細の表示や編集ができます。ネイティブアプリのような操作感を提供します。
- **検索ダイアログ**: モバイルでは、検索アイコンをクリックすると全画面の検索ダイアログが開き、大きな検索結果が表示されます。

## カスタマイズ

`customMobileEventRenderer` オプションを使用して、モバイル用イベントドロワー/ダイアログにカスタムコンポーネントを提供できます。

```tsx
const calendar = useCalendarApp({
  // ...
  customMobileEventRenderer: MyCustomMobileDrawer,
});
```

カスタムコンポーネントは、`isOpen`、`onClose`、`draftEvent`、`app` インスタンスなどの props を受け取ります。
