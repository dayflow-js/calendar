# 機能ショーケース

このセクションでは実際に動作するサンプルを交えながら、プロダクトに組み込む際に役立つ機能を紹介しています。

- **ビュー切り替え**：ボタン式・セレクト式の UI を比較しながら最適なナビゲーションを検討できます。
- **イベント詳細のカスタム化**：既定パネルの中身だけ差し替える方法と、フルカスタムモーダルを用意する方法をそれぞれ実演。
- **イベントダイアログ**：作成/編集/削除をまとめた組み込みダイアログのワークフローを確認。
- **サイドバー**：複数カレンダーやフィルター操作をカスタム UI へ落とし込むパターンを紹介。
- **ダークモード**：ライト/ダーク/システムの自動切り替え、カレンダー単位での配色上書きをテスト。

左のサイドバーから気になるトピックを選び、コードと動作を見比べてみてください。
