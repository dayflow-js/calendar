# イベントダイアログ

DayFlow には、イベントの作成・編集・削除をまとめて行えるダイアログが標準で組み込まれています。モーダルはポータル経由で描画されるため、他の UI と干渉せずに使えます。

<EventDialogShowcase />

## 主な機能

- **フル CRUD**：タイトル・説明・色・時刻を一か所で編集
- **賢い日付処理**：終日 ↔ 時間指定の切り替え、複数日イベントも自動調整
- **レンジピッカー**：開始/終了時刻を直感的に選択
- **カレンダー色の選択**：`calendarId` に紐づく色をそのまま UI に反映
- **レスポンシブ設計**：モバイルでも快適に操作
- **ポータル描画**：オーバーレイや z-index を気にせず配置可能

## 使い方

`DayFlowCalendar` を描画すれば、イベントクリック時にダイアログが自動で開きます。  
新規作成は空きスロットのダブルクリックやドラッグ（ドラッグプラグイン有効時）から呼び出せます。

```tsx copy
import { DayFlowCalendar } from '@dayflow/core';

<DayFlowCalendar calendar={calendar} />;
```

## 操作のヒント

- **タイトルと色**：上部フィールドからイベント名を更新しつつ、所属カレンダーの色で視覚的に整理。
- **時間管理**：レンジピッカーで分単位まで調整。終日トグルでいつでも切り替え可能。
- **ノート**：詳細テキストエリアに議題や参加リンクなどを記入。
- **クイックアクション**：削除ボタンは確認付きで安全に実行でき、変更は自動保存されます。

## カスタマイズ

標準ダイアログで足りない場合は、[カスタム詳細ダイアログ](/docs-ja/features/custom-detail-dialog) を利用して UI 全体を置き換えられます。

```tsx copy
<DayFlowCalendar
  calendar={calendar}
  customEventDetailDialog={YourDialog}
/>;
```

## 挙動

- バックドロップクリックや `Esc` キーで閉じる
- 常に画面中央に配置され、画面幅に応じて自動でリサイズ
- React Portal により `body` 直下へ描画されるため、他のポップアップと z-index が競合しにくい

このダイアログをベースに開発を進めれば、ゼロからフォームを実装することなく、本質的な業務ロジックに集中できます。
