# キーボードショートカット

DayFlowは、生産性を向上させるためにいくつかのキーボードショートカットを提供しています。

## ナビゲーション

| ショートカット | アクション |
| :--- | :--- |
| <kbd>←</kbd> | 前へ (日/週/月/年) |
| <kbd>→</kbd> | 次へ (日/週/月/年) |
| <kbd>Cmd/Ctrl</kbd> + <kbd>T</kbd> | 今日へ移動 |

## イベント管理

| ショートカット | アクション |
| :--- | :--- |
| <kbd>Cmd/Ctrl</kbd> + <kbd>N</kbd> | イベントをクイック作成 |
| <kbd>Cmd/Ctrl</kbd> + <kbd>C</kbd> | 選択したイベントをコピー |
| <kbd>Cmd/Ctrl</kbd> + <kbd>X</kbd> | 選択したイベントを切り取り |
| <kbd>Cmd/Ctrl</kbd> + <kbd>V</kbd> | イベントを貼り付け |
| <kbd>Backspace</kbd> / <kbd>Delete</kbd> | 選択したイベントを削除 |

## インターフェース

| ショートカット | アクション |
| :--- | :--- |
| <kbd>Cmd/Ctrl</kbd> + <kbd>F</kbd> | 検索にフォーカス |
| <kbd>Esc</kbd> | コンテキストメニュー/ダイアログ/詳細パネルを閉じる |
| <kbd>Tab</kbd> | 現在のビュー内のイベントをナビゲート |
| <kbd>Shift</kbd> + <kbd>Tab</kbd> | イベントを逆順にナビゲート |
