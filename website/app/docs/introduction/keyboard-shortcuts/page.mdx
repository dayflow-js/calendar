# Keyboard Shortcuts

DayFlow provides several keyboard shortcuts to improve your productivity.

## Navigation

| Shortcut                           | Action                         |
| :--------------------------------- | :----------------------------- |
| <kbd>←</kbd>                       | Previous (Day/Week/Month/Year) |
| <kbd>→</kbd>                       | Next (Day/Week/Month/Year)     |
| <kbd>Cmd/Ctrl</kbd> + <kbd>T</kbd> | Go to Today                    |

## Event Management

| Shortcut                                 | Action                |
| :--------------------------------------- | :-------------------- |
| <kbd>Cmd/Ctrl</kbd> + <kbd>N</kbd>       | Quick Create Event    |
| <kbd>Cmd/Ctrl</kbd> + <kbd>C</kbd>       | Copy selected event   |
| <kbd>Cmd/Ctrl</kbd> + <kbd>X</kbd>       | Cut selected event    |
| <kbd>Cmd/Ctrl</kbd> + <kbd>V</kbd>       | Paste event           |
| <kbd>Cmd/Ctrl</kbd> + <kbd>Z</kbd>       | Undo last action      |
| <kbd>Backspace</kbd> / <kbd>Delete</kbd> | Delete selected event |

## Interface

| Shortcut                           | Action                                      |
| :--------------------------------- | :------------------------------------------ |
| <kbd>Cmd/Ctrl</kbd> + <kbd>F</kbd> | Focus Search                                |
| <kbd>Esc</kbd>                     | Close Context Menu / Dialog / Detail Panel  |
| <kbd>Tab</kbd>                     | Navigate through events in the current view |
| <kbd>Shift</kbd> + <kbd>Tab</kbd>  | Navigate through events in reverse order    |
