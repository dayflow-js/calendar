# 时区列表

DayFlow 提供了一个 `TimeZone` 枚举，使您可以更轻松地指定辅助时区，而无需记住确切的 IANA 时区字符串。

## 使用方法

```tsx
import { TimeZone } from '@dayflow/core';

// 在视图配置中
createDayView({
  secondaryTimeZone: TimeZone.SHANGHAI,
});
```
