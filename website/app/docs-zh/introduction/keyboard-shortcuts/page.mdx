# 键盘快捷键

DayFlow 提供了一系列键盘快捷键以提高您的效率。

## 导航

| 快捷键 | 操作 |
| :--- | :--- |
| <kbd>←</kbd> | 上一个 (日/周/月/年) |
| <kbd>→</kbd> | 下一个 (日/周/月/年) |
| <kbd>Cmd/Ctrl</kbd> + <kbd>T</kbd> | 跳转到今天 |

## 事件管理

| 快捷键 | 操作 |
| :--- | :--- |
| <kbd>Cmd/Ctrl</kbd> + <kbd>N</kbd> | 快速创建事件 |
| <kbd>Cmd/Ctrl</kbd> + <kbd>C</kbd> | 复制选中的事件 |
| <kbd>Cmd/Ctrl</kbd> + <kbd>X</kbd> | 剪切选中的事件 |
| <kbd>Cmd/Ctrl</kbd> + <kbd>V</kbd> | 粘贴事件 |
| <kbd>Cmd/Ctrl</kbd> + <kbd>Z</kbd> | 撤销上一步操作 |
| <kbd>Backspace</kbd> / <kbd>Delete</kbd> | 删除选中的事件 |

## 界面交互

| 快捷键 | 操作 |
| :--- | :--- |
| <kbd>Cmd/Ctrl</kbd> + <kbd>F</kbd> | 聚焦搜索框 |
| <kbd>Esc</kbd> | 关闭右键菜单 / 对话框 / 详情面板 |
| <kbd>Tab</kbd> | 在当前视图的事件中循环导航 |
| <kbd>Shift</kbd> + <kbd>Tab</kbd> | 反向循环导航事件 |
