# 移动端支持

DayFlow 提供内置移动端支持。日历会自动调整其布局和交互方式，以便在较小屏幕上提供流畅的体验。

## 响应式布局

所有视图（日、周、月）都经过响应式设计：

- **月视图**：隐藏事件时间并使用紧凑的“更多事件”指示器。它还对事件标题采用渐变淡出截断，以最大化可读性。
- **周和日视图**：自动缩小时间列并简化日期标签，以适应较窄的宽度。
- **侧边栏**：侧边栏在移动端自动折叠以节省空间，但仍可通过切换按钮访问。
- **页眉**：视图切换器在移动端自动切换到 `select` 模式，以防止拥挤。

## 移动端交互

DayFlow 引入了移动端特有的交互方式，让管理事件变得简单：

- **长按移动**：在移动端，按住事件 500 毫秒即可启动拖放移动。
- **调整大小指示器**：选中事件时（通过长按或点击），角落会出现大尺寸的触控友好手柄，便于调整大小。
- **事件抽屉**：点击事件会打开一个底部分享栏（Drawer）样式的抽屉，用于查看或编辑详情，提供原生应用般的感受。
- **搜索对话框**：在移动端，点击搜索图标会打开一个全屏搜索对话框，显示大尺寸的搜索结果。

## 自定义

您可以使用 `customMobileEventRenderer` 选项为移动端事件抽屉/对话框提供自定义组件。

```tsx
const calendar = useCalendarApp({
  // ...
  customMobileEventRenderer: MyCustomMobileDrawer,
});
```

自定义组件将接收 `isOpen`、`onClose`、`draftEvent` 和 `app` 实例等 props。
